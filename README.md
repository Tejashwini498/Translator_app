# 🌍 Translator App

A real-time translation application that breaks down language barriers. Translate text between multiple languages instantly with a clean, intuitive interface.

## Features

- **Real-time Translation** – Translate as you type
- **100+ Languages** – Includes English, Spanish, French, German, Chinese, Japanese, Arabic, Hindi, and more
- **Auto Language Detection** – No need to select the source language manually
- **Text-to-Speech** – Listen to pronunciation of translated text
- **Copy & Share** – One-click copy to clipboard or share with others
- **Translation History** – Keep track of your past translations
- **Dark / Light Mode** – Comfortable viewing in any environment

## Tech Stack

- **Frontend:** React / Next.js
- **Styling:** Tailwind CSS
- **Translation API:** Google Cloud Translation / DeepL / LibreTranslate
- **State Management:** Redux Toolkit / Context API
